# Sea Glass Jewelry – Website Design & Product Plan

## Project Overview

This project is an ecommerce website for a handmade **sea glass jewelry brand** based in Thailand.

The jewelry is made from **authentic sea glass collected by hand on Thai beaches**.  
Each piece is unique and naturally shaped by the ocean over many years.

The goal of the website is to showcase these pieces in a **minimalist, elegant, and highly visual experience** that highlights:

- craftsmanship
- uniqueness
- ocean origin
- ecological / recycled nature of sea glass

The website should feel like a mix between:

- a **handmade jewelry boutique**
- an **art gallery**
- a **coastal lifestyle brand**

The emphasis should be on **large visuals, calm design, and product storytelling**.

---

# Core Brand Values

### Handmade

Every jewelry piece is handcrafted from sea glass collected by hand.

### Unique

Each sea glass piece is naturally shaped by the ocean and **no two pieces are identical**.

### Eco-conscious

Sea glass is recycled ocean glass polished naturally by the sea.

### Ocean Story

Every piece carries a story of the ocean and time.

---

# Business Model

The brand releases jewelry through **weekly drops**.

Each drop contains a limited number of pieces.

Example:

Drop #05  
12 unique pieces

When items sell out, they are gone permanently.

The website should highlight:

- the **latest drop**
- **limited availability**
- **next drop countdown**

---

# Design Direction

The website should evoke:

- ocean
- beaches
- sea glass
- calmness
- craftsmanship
- minimal luxury

Avoid generic ecommerce styles.

The design should feel like a **premium handmade jewelry brand**.

---

# Visual Style

The overall visual style should combine:

- coastal minimalism
- Scandinavian design
- Japanese minimalism
- modern luxury jewelry aesthetics

---

# Typography

The website should combine:

**Elegant serif for headings**

Examples:

- Canela
- Playfair Display

**Clean sans-serif for body text**

Examples:

- Inter
- Helvetica Neue

Typography should feel elegant but simple.

---

# UI Style

The UI should use **subtle glassmorphism** inspired by sea glass.

Important:

Do NOT use heavy or flashy glass effects.

Instead aim for:

- frosted glass surfaces
- soft transparency
- natural light reflections
- subtle blur

The feeling should resemble **polished sea glass**.

---

# Layout Principles

The layout should follow:

- large product images
- generous whitespace
- calm spacing
- minimal UI clutter
- strong visual hierarchy

Products should feel like **gallery objects**.
